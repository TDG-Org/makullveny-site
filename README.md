# Makullveny Site

Public one-page landing page for Makullveny support/account verification.

This repo intentionally does not contain the private Makullveny desktop app source code. It only explains the project at a high level so payment providers and supporters can understand what Makullveny is.

## GitHub Pages

The site is designed to publish from the repository root through GitHub Pages.

Expected URL after Pages finishes deploying:

`https://tdg-org.github.io/makullveny-site/`
